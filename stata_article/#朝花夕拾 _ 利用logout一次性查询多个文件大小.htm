Stata 的 checksum 命令，在查询文件大小时，每次只能查询一个文件，比较耗时。如果有大量的文件要查询其大小，建议使用 dir 并配合 logout 。

dir 是 stata 的一个 DOS 命令，能够给出文件夹中的文件列表，用户可以运行如下的命令：

cd d:/

dir

logout 是 stata 用户编写的一个外部命令，能够将命令行生成的表格输出为 Word, Excel, LaTex, 或 Stata datafile 中，用户可以运行如下程序：

ssc install logout // 下载 logout 命令包

sysuse auto,clear

logout, save(myfile) word excel replace:pwcorr price headroom weight length

logout + dir 一次性查看所有文件的大小：

第一步：新建文件夹 d:/temp/

clear

capture mkdir d:/temp/

第二步：在该文件夹下面生成 10 个文件

forval i = 1(1) 10{

disp `i'

clear

local OBS = `i'*100

set obs `OBS'

gen x1 = uniform()

gen x2 = _n

gen x3 = rnormal()

save d:/temp/`i',replace

}

第三步：更改 stata 默认路径的设置

cd d:/temp/

第四步：将文件信息输出到 d:/mystuff.dta 文件中

logout, save(d:/mystuff) dta replace: dir

第五步：查看文件大小

use d:/mystuff, clear

drop in 1/2

rename v5 filename

rename v1 size

br
