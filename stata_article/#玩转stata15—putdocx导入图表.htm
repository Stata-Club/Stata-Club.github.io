之前的推文《 玩转stata15—putdocx生成word文档 》详细介绍了 putdocx 的强大功能，用命令行生成 word 文档。而在作实证的时候，除了检验结果需要导入 word， 图表也有这个需要，那就接着请 putdocx 来帮忙吧！


关闭文档

由于使用putdocx命令去修改一个已经打开的.docx文件，Stata15将会报错，因此我们用命令关闭已经打开的word文档以防报错。

! taskkill /F /IM WINWORD.EXE /T


构造图表

用系统自带的auto数据，依据变量mpg（消耗每加仑油能走的公里数）的频数构造直方图。并保存在指定路径下，命名为hist.png文件。

clear

set more off

cd C:\Users\Dell\Desktop

sysuse auto

histogram mpg

graph export hist.png,replace


导入表格

putdocx begin //声明我们要开始putdocx了

putdocx paragraph,halign(center) // 写入一段，将段落内容居中

putdocx text ("每加仑耗油量直方图"), bold linebreak //输入加粗的文本"每加仑耗油量直方图"，并在结尾处换行。

putdocx paragraph,halign(center) // 写入一段，将段落内容居中

putdocx image hist.png, width(4) //导入图表并设置文档宽度为4

putdocx save hist.docx,replace


有两句话不知当讲不当讲


1. putdocx image 后的 option 选项，能够让我们的图表导入更加精确细化。

width ：设置图像的宽度。如果宽度大于文档的页面宽度，则使用文档宽度。如果未指定width（），则使用默认大小。这里默认值由文档的图像信息和页面宽度决定。

height ：设置图像的高度。如果没有指定height（），则图像的高度由图像的宽度和宽高比决定。

link ：链接指定将图像文件名的链接插入文档。如果图像被链接，则引用的文件必须存在，以便文档可以显示图像。

linebreak ：指定在新图像之 后添加换行符。此选项可以多次指定，以在新图像之后添加多个换行符。

2.putdocx image 的图片后缀名可以是便携式网络图形（.png），JPEG（.jpg），增强型元文件（.emf）或标记的图像文件格式（.tif）文件。
