一、下载sublime text 3

在百度中搜索 sublime text 3 ，进入sublime text官网，官网链接是 http://www.sublimetext.com/ 。下载界面如图所示：

点击 Download for Windows 按钮，下载sublime text 3，需要 注意 的是这里下载的sublime text 3需要 64位系统 。下载好之后，点击 安装 ，需要注意的是这里 Add to explorer context menu 前面需要 打钩 。

这样我们就把 sublime text 3 下载好了。

二、配置sublime text 3三步走

第一步 ： 双击 sublime text 3图标，点击 Tools>install Package Control

第二步 ： Preferences>PackageControl>下拉列表:Install Package 中输入 Pywin32 ，点击下载。

同理可以下载stataeditor。 Preferences>PackageControl>下拉列表:Install Package 中输入 Stataeditor ，点击下载。

第三步 ：把 Preferences>Package Settings>StataEditor>Settings>Default 里的文本 复制 到 Preferences>PackageSettings>StataEditor>Settings>User 里。需要 改动 三处。

1、“C:/Program Files (x86)/Stata13/StataMP-64.exe”改成你的 stata所在的位置 加你的 stata版本 。

2、“"stata_version": 13,”改成“"stata_version": 15 ,”

3、 “windows-1252”改成“ utf-8 ”

三、配置stata两步走

第一步 ：点开stata，在stata的 Do-file Editor > Edit > Preferences 的 Advanced 一栏中，把Edit do-files opened from Windows instead of executing them前面的 勾取消 。点击 apply 再点击 OK 。

第二步 ： 右键Stata创建快捷方式 ，右击stata快捷方式，左击 属性 。在目标中 添加/Register ,需要注意的是下图“D:\stata15\stataMP-64.exe”与“/Register”之间需要 空格隔开 。注意这里是“ / ”。右击快捷方式，选择“ 以管理员身份运行 ”。
