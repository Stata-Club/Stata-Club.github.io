stata绘图语言一直变幻莫测，这也让我们这些编程渣渣异常头痛，今天小编就给大家展示一下图形编辑器的花样玩法，一个新工具 gr_edit 命令，有了该神器我们可以在stata的图形编辑器中任意更改，而且还可以生成我们想要的dofile文件命令。接下来小编就通过以往一个读者对mrgraph绘图的x标签字体大小的设置用命令来写问题为例，来介绍gr_edit的好处

安装mrgraph命令：

ssc install mrgraph,replace

首先我们通过调用网页数据：

use http://fmwww.bc.edu/RePEc/bocode/d/drugs.dta

通过使用mrgraph命令进行绘图：

mrgraph bar crime1-crime5, include response(2 3) sort width(15) title(Criminal experiences (as a victim)) ylabel(,angle(0))

对于五种犯罪标签我们如何字体大小一般我们使用图形编辑器，首先通过点击左图红框图形进入图形编辑环境，然后通过双击x轴标签可以得到右下图：

通过该步骤进行修改标签字体大小，可得到：

可以观测红框区域内标签字体大小的变换

如果我们想将此流程变为程序，我们需要先点击红框图形进入编辑环境， 其次我们要事先点击右上角的红点

然后再按照上述流程图改变字体大小，最后再点击右上框的红色按钮即可 （切记红色按钮总共点击两次，编辑前一次编辑后一次）

通过该步骤我们将命令保存到自己文件储存的位置，只需要通过记事本打开即可

我们通过将红框中的编码放到dofile中在编码前加上gr_edit即可

程序变为：

mrgraph bar crime1-crime5, include response(2 3) sort width(15) title(Criminal experiences (as a victim)) ylabel(,angle(0))

gr_edit grpaxis.edit_tick  1 8.66337 `"`"use a weapon"' `"against someone"'"', tickset(major) editstyle(tickstyle(textstyle(size(medlarge))) )

图形即可改变标签大小

希望通过该网络数据以及mrgraph绘图为例来介绍 gr_edit 生成图形命令的伟大，另外如果有对mrgraph多项选择绘图的命令感兴趣的可以自行help mrgraph，里面会对mrgraph以及mrtab详细介绍。同时也希望gr_edit能够帮助更多stata学者绘制更多完美的图形。
